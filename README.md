## Assumptions & Limitations

This project uses a limited three-year financial dataset extracted from SEC Form 10-K filings.
As a result, the forecasting model is designed as an interpretable baseline rather than a high-accuracy predictive system.

Key limitations include:
- Small dataset size (three observations)
- Linear trend assumption
- No external macroeconomic or real-time data
- Forecasts intended for directional insight only

Future enhancements may include extended datasets, time-series modeling, and LLM-based chatbot integration.
